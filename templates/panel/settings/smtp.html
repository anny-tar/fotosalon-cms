📧 Яндекс Почта
  1. Откройте id.yandex.ru/security
  2. Найдите раздел «Пароли приложений»
  3. Нажмите «Создать новый пароль» → выберите тип «Почта»
  4. Введите название (например «ФотоСалон») и скопируйте пароль
  5. В настройках почты включите доступ по SMTP:
    Все настройки → Почтовые программы → С сервера imap.yandex.ru по протоколу IMAP
Настройки:
SMTP серверsmtp.yandex.ru
Порт587
Логинваш_email@yandex.ru
Парольпароль приложения
Адрес отправителяваш_email@yandex.ru
📧 Gmail (Google)

Gmail требует двухфакторную аутентификацию и пароль приложения.

  1. Включите двухфакторную аутентификацию на myaccount.google.com/security
  2. Перейдите в myaccount.google.com/apppasswords
  3. Выберите приложение «Почта» и устройство «Другое»
  4. Введите название (например «ФотоСалон») и нажмите «Создать»
  5. Скопируйте 16-значный пароль — он показывается только один раз
Настройки:
SMTP серверsmtp.gmail.com
Порт587
Логинваш_email@gmail.com
Парольпароль приложения (16 знаков)
Адрес отправителяваш_email@gmail.com
📧 Mail.ru

Mail.ru также требует создать отдельный пароль для приложения.

  1. Перейдите в account.mail.ru/user/security/passwords
  2. Нажмите «Добавить» в разделе «Пароли для внешних приложений»
  3. Введите название (например «ФотоСалон»)
  4. Скопируйте сгенерированный пароль
  5. В настройках почты разрешите доступ по SMTP:
    Настройки → Безопасность → Включить доступ по протоколу POP3 и SMTP
Настройки:
SMTP серверsmtp.mail.ru
Порт587
Логинваш_email@mail.ru
Парольпароль приложения
Адрес отправителяваш_email@mail.ru
